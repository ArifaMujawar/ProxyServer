# ProxyServer
To Run the ProxyServer
  - Generate yelp Api key from [here](https://www.yelp.com/developers)
     - Select yelp Fusion
     - Press Get Started Button
  
 - Save the Api key in .env file of Proxy Server project.
 - Install npm packages of the project
    `npm install`
 - Run the application 
    `npm start`
